# 💌 更新日志

## 🌱 v1.0.0 - 2026-06-21

每日分享 1.0.0 初版发布。这个版本以“在合适的时刻分享日常”为核心，把文案生成、真实来源、媒体生成、QQ 空间、定时任务和 Pages 仪表盘整理成一套完整体验。

### ✨ 初版能力

- 📝 分享类型：支持 `问候`、`新闻`、`心情`、`知识`、`推荐` 五类分享。
- 🔍 真实来源：新闻优先使用源站摘要、正文和链接；知识、推荐和新闻背景可结合 AstrBot 已配置的 Tavily / Brave 联网检索。
- 📰 新闻能力：支持多热搜源、每天 60s 读世界、AI 资讯快报、新闻长图、短链接和原文链接追问。
- 🎨 媒体生成：配图、视频和语音默认调用 `astrbot_plugin_daily_life`；视频使用配图生成，图片和视频不会重复混发。
- 🌐 QQ 空间：支持发布文字、图片或单视频说说，查看动态与详情，并进行点赞、评论、删除和自动互动。
- ⏰ 定时分享：支持固定时间、随机时段、智能定时、高级 Cron、独立目标定时和独立类型序列。
- 🧠 场景上下文：可结合生活状态、聊天历史、群聊热度，并复用 `astrbot_plugin_daily_life` 的 MemOS 记录，让分享内容更贴近当前场景。
- 🖥️ Pages 仪表盘：提供状态总览、手动触发、设置管理、动态与媒体、历史记录、失败记录、日历和 QQ 空间面板。
